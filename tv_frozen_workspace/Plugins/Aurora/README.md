Serialize, Initialize, Deserialize and Import Server Maps to and from JSON or PostgreSQL Database. Check out the parent project [Target Vector](https://github.com/Voidware-Prohibited/TargetVector).

_Aurora is under heavy development, many features may not be finished or production-ready_

# Features

- Initialize, Deserialize and Import Server Maps from JSON or Database.
- Serialze Server Maps to JSON or Database.
- Database Single-Source-of-Truth

# Installation

Clone or Download into a folder named _Aurora_ in your existing projects Plugins folder.
